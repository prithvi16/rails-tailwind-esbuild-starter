# Rails 6.1, TailwindCSS 2, Esbuild boilerplate

### Instructions

- Replace `starter` with `your-app-name`
- Replace `Starter` with `YourAppName`

```
  bundle install
  yarn
```

- To run app
```
  # terminal 1
  yarn build --watch
  # terminal 2
  rails s
```

No webpack config files were harmed in this repository.
